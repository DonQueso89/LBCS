# LBCS

## LBCS client setup
- install the expo-cli
```bash
npm install -g expo-cli
```

## LBCS server setup
- install poetry using `pip`
```python
pip3 install poetry
```
- use poetry to install the requirements. `cd` into the `ledserver` repo and type:
```python
poetry install 
```
this should install all the dependencies
